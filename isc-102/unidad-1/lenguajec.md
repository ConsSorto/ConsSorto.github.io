## 1: Lenguaje C - Historia y Estructura

Para entender C, debemos conocer su evolución. C surgió de los lenguajes BCPL y B en la década de 1970, siendo **Dennis Ritchie** quien lo creó originalmente en los Laboratorios Bell para implementar el sistema operativo UNIX (Capítulo 1, Sección 1.14). Se le considera un lenguaje de "nivel medio" porque permite un control cercano al hardware pero con la legibilidad de un lenguaje de alto nivel (Capítulo 3, Introducción).

Un programa típico en C se organiza mediante **directivas del preprocesador** como `#include` (Capítulo 3, Sección 3.1), declaraciones globales, la **función principal `main()`** y definiciones de otras funciones necesarias para resolver el problema (Capítulo 3, Sección 3.1). 
Los comentarios, encerrados entre `/* ... */`, son esenciales para documentar el código, aunque el compilador los ignora por completo (Capítulo 3, Sección 3.1).
